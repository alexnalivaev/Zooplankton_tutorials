# Zooplankton_tutorials
Exemples de codes simples sous Python, pour analyser les données CMIP6, et pour tracer des cartes.

Le dossier est en cours de création: pour le moment, on a des exemples de codes simples pour tracer des cartes sous Python, avec matplotlib et cartopy.
Ojectif final: le dossier devrait contenir:
1) des exemples de codes bash utilisant CDO (climate data operators): traitement et formattage des données CMIP6.
2) Codes Python pour analyser les projections de biomasse de zooplancton et calculer des incertitudes intermodèle.
3) Codes plots (en cours).
